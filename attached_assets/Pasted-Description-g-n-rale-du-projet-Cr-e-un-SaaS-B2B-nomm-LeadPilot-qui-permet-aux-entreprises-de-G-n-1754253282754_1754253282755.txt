Description générale du projet
Crée un SaaS B2B nommé LeadPilot qui permet aux entreprises de :

Générer automatiquement des leads qualifiés via l’API Apollo

Filtrer et scorer les leads avec l’IA OpenAI GPT-4o

Vérifier les emails et enrichir les données (nom, entreprise, secteur)

Lancer des campagnes d’emails cold avec 30 modèles prédéfinis en français

Générer des variations d’emails via IA (modifiables par l’utilisateur)

Automatiser des séquences multi-étapes (3 étapes pour Pro, 5 étapes pour Growth)

Suivre les statistiques (taux ouverture, clic, réponse, RDV bookés)

Fournir un calendrier intégré pour que les leads puissent réserver des rendez-vous

Gérer les abonnements et quotas via Stripe (plans Free, Starter, Pro, Growth)

Préparer une future option “Closing” (délégation RDV closers, visible mais verrouillée)

Plans et quotas (mensuels)

Pas d’A/B testing

Growth (299€/mois)

Fonctionnalités clés
Génération de leads via Apollo API

Filtrage et scoring avec GPT-4o

Stockage des leads dans Supabase (PostgreSQL)

30 modèles d’emails intégrés (selon plan)

Génération de variations IA

Séquences multi-étapes automatisées

Suivi statistique complet

Calendrier de booking intégré

Gestion abonnements & quotas via Stripe

Automatisations mensuelles (reset quotas, relances, webhooks)

Architecture technique
Backend : FastAPI (Python)

Base : Supabase (PostgreSQL)

IA : OpenAI GPT-4o

Lead source : Apollo API

Emailing : SMTP via compte client + fallback SendGrid

Paiement : Stripe (checkout + webhooks)

Migrations DB : Alembic

Config : .env pour API keys

Tables principales : users, templates, leads, campaigns, emails, bookings, payments
